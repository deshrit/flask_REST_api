# flask_REST_api
RESTful service using flask using file

## For demo

**Clone this repo**
```
git clone git@github.com:deshrit/flask_REST_api.git
```

**Change dir**
```
cd flask_REST_api
```

**Run server**
```
flask --app rest_api run
```

**Open any browser and search**
```
http://localhost:5000/api/users
```
